# Sudoku Generator
Rayhan Alwi

A java program that generates a unique sudoku puzzle, solvable in terminal.

*Run the program by downloading the latest Java compiler and running the Main.java file.*

The sudoku board is generated through a backtracking, recursive algorithm that repeatedly places numbers and checks if that number is a valid sudoku number. The board is first seeded with 3 3x3 squares, the top left, center, and bottom right squares. The algorithm builds off of these squares.

Files:
* README.md - this file
* Main.java - the main java file, run this.
* Sudoku.java - contains the sudoku logic
